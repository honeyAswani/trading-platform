# вҡҷпёҸ Getting Started

## Clone the Repository

```bash
git clone https://github.com/honeyAswani/trading-platform.git
```

```bash
cd trading-platform
```

---

## Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Dashboard

```bash
cd dashboard
npm install
```

### Backend

```bash
cd backend
npm install
```

---

# рҹ”җ Environment Variables
